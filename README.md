## DeepTraffic-Grid: Grid-Based Deep Learning Models for Urban Traffic Prediction

## Dependencies
Working environment and major dependencies:
* Ubuntu 16.04.6 LTS
* Python 3 (>= 3.5; Anaconda Distribution)
* NumPy (>= 1.11.0)
* pandas (>= 0.18.0)
* TensorFlow/tensorflow-gpu (= 1.1x.0)
* Keras (>= 2.0.8)

## Datasets
* TaxiBJ
* BikeNYC-I
* BikeNYC-II
* TaxiNYC

## Models
* CNN
* ConvLSTM
* STResNet
* PCRN
* DMVST-Net
* DeepSTN+
* STDN
